# The guide of Rijeka

Final paper - Project scripts
Author: Klara Popovic

The Guide of Rijeka is a turist mobile app for android phones made in 2019 for my final paper in university.

SPECIFICATIONS:

Type: Android mobile app
Engine: Android Studio
Language: Java
Genre: Informational, Turist, Educational
Platform: Android phone

ABOUT
The Guide of Rijeka is an Android application for getting to know the city of Rijeka.
It contains attractions sorted into groups where each attraction contains a picture, short description and/or history behind it and a Google Maps link that guides user to its location.
